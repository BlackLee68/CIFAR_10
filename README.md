# CIFAR_10
CIFAR10是一个图像数据库，里面包含了十个种类60000张图片，每个种类平均6000张左右。

这里采用CNN网络来对这个数据库进行测试。

CIFAR10：http://www.cs.toronto.edu/~kriz/cifar.html

这里使用的是dataset的二进制版本，你可以在这个根文件夹的另一个文件夹里找到使用Python版本的dataset。

Notice: 本方法是基于tensorflow。
